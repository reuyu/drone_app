# UI 개발용 임시 폴더

이 폴더는 UI 작업을 위한 임시 개발 환경입니다.

## ✨ 최신 변경사항 (2026-01-01)

### 주요 개선사항
- ✅ 모바일 최적화 UI 적용
- ✅ Detection Log 재구성 (오늘 n개 탐지, 확률 색상 표시)
- ✅ 지역명 기능 추가 (위도/경도 → 지역명 변환)
- ✅ Detection Card 레이아웃 변경 (확률 → 시간 → 지역)
- ✅ HistoryPage 재구성 (날짜, 시간, 지역, 드론, 풍속, 온도)
- ✅ 드론 목록에 오늘 탐지 사건 수 표시
- ✅ 하단 네비게이션 색상 변경 (검은색 기본, 활성 탭 볼드체)
- ✅ 연결 상태 표시 (MonitorPage 헤더 우측)
- ✅ 토스 스타일 클릭 피드백

**자세한 변경사항은 [CHANGELOG.md](./CHANGELOG.md) 참고**

## 특징

- **Mock 데이터 사용**: 서버 없이도 UI를 테스트할 수 있습니다
- **임의 드론 데이터 포함**: 3개의 샘플 드론이 미리 등록되어 있습니다
- **독립 실행**: 원본 프로젝트와 분리되어 있어 안전하게 UI를 수정할 수 있습니다

## 포함된 Mock 데이터

### 드론 목록
- `drone_alpha` - 활성 상태, 화재 감지 로그 7개
- `drone_beta` - 활성 상태, 화재 감지 로그 5개  
- `drone_gamma` - 오프라인 상태, 로그 없음

### 화재 감지 로그
- 각 드론별로 다양한 확률의 감지 로그 포함
- 오늘 날짜 이벤트 포함
- 이미지 URL은 placeholder 사용

## 실행 방법

```bash
cd frontend_ui_dev
npm install
npm run dev
```

브라우저에서 http://localhost:5173 접속

## Mock 데이터 수정

`src/mockData.js` 파일을 수정하여 원하는 드론 데이터를 추가/수정할 수 있습니다.

## 주요 기능

### 1. 드론 목록 (HomePage)
- 드론 카드에 "오늘 n개 탐지" 배지 표시
- 배터리 상태 표시
- 연결 상태 표시

### 2. 모니터링 (MonitorPage)
- GPS 정보에 지역명 표시
- Detection Log: "오늘 n개 탐지" 형식
- 최근 탐지: 확률(색상), 시간(24시간), 지역, 풍속, 온도
- Detection Cards: 확률 → 시간 → 지역 순서
- 연결 상태 점 표시 (헤더 우측)

### 3. 히스토리 (HistoryPage)
- 날짜, 시간(24시간), 지역, 탐지 드론, 풍속, 온도 순서
- 확률 배지 표시

### 4. 지역명 기능
- `src/utils/location.js`에서 위도/경도 → 지역명 변환
- 서울 지역 기반 간단한 매핑 (실제로는 API 연동 필요)

## 주의사항

- 이 폴더의 변경사항은 원본 프로젝트에 영향을 주지 않습니다
- UI 작업이 완료되면 변경사항을 원본 프로젝트에 반영해야 합니다
