## 📋 이관 방법

### 방법 1: 압축 파일로 넘기기 (권장)

#### Step 1: 불필요한 파일 제외
```powershell
# node_modules 제외하고 압축
# 또는 압축 프로그램에서 node_modules 폴더 제외 선택
```

#### Step 2: 압축
```
DroneUI/
└── firedrone.zip  (node_modules 제외)
```

#### Step 3: 받는 사람이 해야 할 일
```bash
# 압축 해제
unzip firedrone.zip

# 의존성 설치
cd firedrone
npm install

# 실행
npx expo start
```

---

### 방법 2: Git 사용 (권장 - 버전 관리)

#### 현재 프로젝트에서
```bash
cd firedrone

# .gitignore 확인 (node_modules가 제외되어 있는지)
cat .gitignore

# Git 초기화 (아직 안 했다면)
git init

# 파일 추가
git add .

# 커밋
git commit -m "Initial commit: FireDrone mobile app"

# 원격 저장소에 푸시 (GitHub, GitLab 등)
git remote add origin [저장소 URL]
git push -u origin main
```

#### 받는 사람이 해야 할 일
```bash
# 클론
git clone [저장소 URL]

# 의존성 설치
cd firedrone
npm install

# 실행
npx expo start
```
