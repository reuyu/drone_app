# 📱 앱 커넥션 문제 해결 가이드

## 문제: "앱 커넥션이 없다" 오류

### 해결 방법

#### 1. Expo Go 앱 확인
- **iOS**: App Store에서 "Expo Go" 설치
- **Android**: Play Store에서 "Expo Go" 설치

#### 2. 같은 네트워크 확인
- **컴퓨터와 휴대폰이 같은 Wi-Fi에 연결되어 있어야 합니다**
- 모바일 데이터 사용 시 연결되지 않을 수 있습니다

#### 3. 방화벽 확인
- Windows 방화벽이 Expo 서버를 차단하지 않는지 확인
- 방화벽 예외 추가:
  ```
  포트: 8081, 19000, 19001
  ```

#### 4. QR 코드 스캔
1. 터미널에 표시된 **QR 코드** 확인
2. **Expo Go 앱** 실행
3. **"Scan QR code"** 선택
4. QR 코드 스캔

#### 5. 수동 연결 (QR 코드가 안 보일 때)
1. 터미널에서 **`s` 키** 입력 (서버 URL 표시)
2. Expo Go 앱에서 **"Enter URL manually"** 선택
3. 표시된 URL 입력 (예: `exp://192.168.0.100:8081`)

#### 6. 터널 모드 사용 (같은 네트워크가 아닐 때)
```bash
npx expo start --tunnel
```
- 더 느릴 수 있지만 다른 네트워크에서도 작동합니다

#### 7. LAN 모드 사용
```bash
npx expo start --lan
```

#### 8. 캐시 클리어 후 재시작
```bash
npx expo start --clear
```

## 추가 도움말

### 터미널 명령어
- **`r`**: 앱 새로고침
- **`m`**: 개발자 메뉴 열기
- **`s`**: 서버 URL 표시
- **`w`**: 웹 브라우저에서 열기
- **`a`**: Android 에뮬레이터에서 열기
- **`i`**: iOS 시뮬레이터에서 열기 (Mac만)

### 문제가 계속되면
1. Expo Go 앱 재시작
2. 컴퓨터 재부팅
3. Wi-Fi 재연결
4. `npm install` 재실행
